mobf01:用户登陆
功能说明： 用户登陆
程序映射地址：
返回值例子： mobf01.xml | mobf01.txt
参数说明：
CODE NAME 是否必填 备注说明
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
iphonetoken Iphone的token值 选填
account 用户登录账户 必填 邮箱或手机号
password 密码 必填
业务逻辑说明： 1.检验账户格式。
2.根据userid、sessionid登录校验。
3.将用户手机信息写入user_mobile_info（用户移动终端信息表）（包含：手机mac地址、手机登录后的SessionId值、Iphone的token值）。
