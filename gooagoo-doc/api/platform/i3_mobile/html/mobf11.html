接口 mobf11 : 获取APP最新版本
功能说明： 检查更新gooagoo APP最新版本
程序映射地址：
返回值例子： mobf11.xml | mobf11.txt
参数说明：
CODE NAME 是否必填 备注说明
versioncode 程序版本 必填
appcode APP识别码 必填
mobiletype 系统类型 必填 1-Android 2-IOS
业务逻辑说明： 1.检查更新版本分为两种：一种是系统自动校验更新手机APP版本，下载最新的APP。另一种是，用户进入用户设置界面，手动点击【检测新版本】按钮，联网检查Gooagoo App最新的版本代码versionCode、版本名称versionName、最新app的apk包下载地址，与当前用户手机安装的App版本进行对比。如果最新app的versionCode大于当前版本，则前往apk包的下载地址进行下载更新最新的APP。
