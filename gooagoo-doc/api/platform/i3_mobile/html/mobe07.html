mobe07:通过桌号获取点菜单信息
功能说明： 通过桌号获取点菜单信息
程序映射地址：
返回值例子： mobe07.xml | mobe07.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
tableno2d 桌号二维码 必填
业务逻辑说明： 1.根据userid、sessionid校验登录状态；
2.解析code2d获取tablename、roomname、entityid。
3.根据步骤2查询order_info（订单信息）、order_detail_info（订单商品详细信息），获取点菜单信息。
4.封装查询结果，返回给手机。
