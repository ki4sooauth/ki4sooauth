mobe10:用户获取账单信息接口(商家服务器)
功能说明： 用户获取账单信息接口(商家服务器)
程序映射地址：
返回值例子： mobe10.xml | mobe10.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
shopid 店铺id 必填
ctimestamp 最大时间戳 必填 最后一次修改时间，同步全部则设置为0
业务逻辑说明： 1.根据userid、sessionid校验登录状态；
2.查询用户order_info（订单信息列表）中bill_type（订单状态）为“3-已结账”的账单信息。（只查询“最后一次修改时间”大于“最大时间戳”的数据，如果“最大时间戳”无值则查询所有数据）。
备注： 本接口用于查询部署在“商家服务器”中“本店账单”的账单列表，返回数据不包含器“Gooagoo"服务器中的账单列表。
