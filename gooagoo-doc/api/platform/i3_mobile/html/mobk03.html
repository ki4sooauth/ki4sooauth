mobk03:已购买过的商品
功能说明： 已购买过的商品。
程序映射地址：
返回值例子： mobk03.xml | mobk03.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填
sessionid session值 必填
mac 手机mac地址 选填
mver 手机程序版本 选填
lid lid 选填
pageindex 页码 必填
pagesize 每页信息显示条数 必填
shopentityid 实体店编号 选填 通过卡查询时购过商品必填
shoptype 商家类别 选填 通过商铺类别查询时必填
业务逻辑说明： 1.通过商家类型或者实体店编号直接查询用户购过商品列表。
