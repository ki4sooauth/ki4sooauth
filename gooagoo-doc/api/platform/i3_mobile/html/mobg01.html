mobg01:基本信息
功能说明： 基本信息
程序映射地址：
返回值例子： mobg01.xml | mobg01.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 选填 用户编号，唯一，通过UUID产生
sessionid sessionid值 选填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
type 同步方式 必填 1-根据syninfo获得同步信息，2-根据entityposition获得同步信息
syninfo 同步信息 选填 json串，格式： { "shopinfo": [ { "shopid": "183F09S65984M12K0IJBFOR5OI6FCVML", "ctimestamp": "2012-01-01 01:01:01" } ], "shoppinglistgoodstype": [ { "ctimestamp": "2012-01-01 01:01:01" } ], "shoptype": [ { "ctimestamp": "2012-01-01 01:01:01" } ] }
containcode 同步范围 必填 对应返回报文中“containcode”，多个时以逗号分隔，例如“A,B,C”;为空时查询所有,A-商家基本信息 B- 购物计划商品分类 信息 C-商家类型信息
业务逻辑说明： 1.根据type查询基本信息。
备注：
基本信息包括：
1、 商家基本信息
a)基本信息shopinfo
b)辅助信息shopprofile
c)实体店基本信息shopentityinfo
d)实体店联系方式shopentitylink
e)实体店Wifi信息shopwifiinfo
f)实体店Gps信息shopgpsinfo
g)服务工具信息shoptoollist
h)发票项目类型invoicetype
I)积分商城地址shopintegralurl
J)接口地址列表shop_interface_info
k)会员卡基本信息membercard
2、 购物计划商品分类shopping_list_goods_type
3、 商家类型shop_type
