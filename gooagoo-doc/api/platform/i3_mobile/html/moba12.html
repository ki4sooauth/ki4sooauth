moba12:用户是否同意发卡
功能说明： 用户是否同意发卡
程序映射地址：
返回值例子： moba12.xml | moba12.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
shopid 店铺id 必填
cardid 卡号 必填
cardtype 卡类型 必填
isagree 是否同意 必填 Y-同意，N-不同意
业务逻辑说明： 1.根据userid、sessionid校验登录状态；
2.如果isagree=Y则根据shopid、cardtype、cardid修改member_of_card表相关信息，并修改member_base_info相关信息。
3.返回操作结果true-成功，false-失败 。
