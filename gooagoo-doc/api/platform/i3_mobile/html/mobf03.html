mobf03:用户注册
功能说明： 用户注册
程序映射地址：
返回值例子： mobf03.xml | mobf03.txt
参数说明：
CODE NAME 是否必填 备注说明
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
email 用户邮箱 必填
password 密码 必填
业务逻辑说明： 1.用户邮箱格式不正确，返回false，提示：邮箱格式有误，请重新输入。
2.用户邮箱已经被注册过，返回false，提示：该邮箱已被使用，请使用其邮箱。
3.邮箱校验通过，返回true，提示用户需激活后使用。
