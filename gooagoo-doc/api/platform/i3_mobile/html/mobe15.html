mobe15:绑定桌号、订单号、用户id
功能说明： 远程订餐到店后将桌号与订单绑定。
程序映射地址：
返回值例子： mobe15.xml | mobe15.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
orderid 订单号 必填
tableno2d 桌号二维码 必填 tableno2d 格式["gooagoo","00",
"实体店编号","桌号","房间号"]
业务逻辑说明： 1.根据orderid查询出订单信息，将tablename、userid写入order_info表，关联订单号、桌号、用户编号。
