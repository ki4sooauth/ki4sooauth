moba01:查询用户会员卡列表
功能说明： 查询与用户关联的会员卡列表
程序映射地址：
返回值例子： moba01.xml | moba01.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户编号 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac 手机mac地址 选填
mver 手机程序版本 选填
lid lid 选填
entityposition 实体店编号 选填 用户当前所在实体店编号
ctimestamp 最大时间戳 必填 手机初次进入则传2000-01-01 00:00:00
pagesize 每页显示信息条数 必填
业务逻辑说明： 1.校验登录状态；
2.根据ctimestamp查询member_of_card表中，“c_time_stamp”大于入参“ctimestamp”的、并且最接近入参“ctimestamp”的一页会员卡列表。
3.如果ctimestamp为2000-01-01 00:00:00，则查询member_of_card表中按时间戳排序最靠后的一页
4.封装查询结果，返回数据。
