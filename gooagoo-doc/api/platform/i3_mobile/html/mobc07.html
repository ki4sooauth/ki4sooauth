mobc07:商家推荐吆喝
功能说明： 商家推荐吆喝
程序映射地址：
返回值例子： mobc07.xml | mobc07.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户编号 选填 用户编号
sessionid sessionid值 选填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
cryoutinfoid 吆喝编号 选填 平台通过此参数返回大于或小于
此"吆喝编号"的吆喝。
pagetype 翻页类型 选填 统一分页规则： p-上一页（查询新数据） ，ID传当页第一条数据，平台查询此ID上一页数据 N-下一页（查询旧数据），ID传当页最后一条数据，平台查询此ID下一页数据 例如： 1.数据正序排列 分页大小为3， 全部数据为123，456，789 当前页数据为456 查询上一页数据：pagetype=P,pageid=4,查询结果为123 查询下一页数据：pagetype=N,pageid=6,查询结果为789 1.数据倒序排列 分页大小为3， 全部数据为987，654，321 当前页数据为654 查询上一页数据：pagetype=P,pageid=6,查询结果为987 查询下一页数据：pagetype=N,pageid=4,查询结果为321
pageindex 页码 必填 当pageindex=1，pagesize=-1时，不分页
pagesize 每页显示信息条数 必填
业务逻辑说明： 1.根据userid查询商家推荐吆喝。
2.推荐吆喝是根据行为分析结果获取。
