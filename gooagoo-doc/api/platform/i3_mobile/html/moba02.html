moba02:手机端删除会员卡接口
功能说明： 手机端删除会员卡。
程序映射地址：
返回值例子： moba02.xml | moba02.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac 手机mac地址 选填
mver 手机程序版本 选填
lid lid 选填
entityposition 实体店编号 选填 用户当前所在实体店编号
scardno 会员卡音频卡号 必填 字符串，其间要用“，”分隔开。
业务逻辑说明： 1.校验登录状态；
2.根据infoid删除对应的member_of_card（会员表），支持一条或多条同时删除。
3.注意这里的删除只是将各个表中的is_del改成Y-已删除。
