接口 mobo02 : 编辑用户收货地址信息
功能说明： 编辑用户收货地址信息
程序映射地址：
返回值例子： mobo02.xml | mobo02.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户编号 必填
sessionid 用户sessionid 必填
consigneeid 收货人信息编号 必填
consigneename 收货人姓名 必填
province 省 必填
city 市 必填
area 区县 必填
address 详细地址 必填
phone 手机号码 选填 手机号码和联系电话,两者二选一必填
telephone 联系电话 选填 手机号码和联系电话,两者二选一必填
isdefault 默认选项 必填 是否默认选项，Y-是，N-否,选是表示该地址为用户的某人收货地址信息
