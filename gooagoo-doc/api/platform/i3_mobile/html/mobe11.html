mobe11:用户评论商品
功能说明： 用户对已购买的商品进行评论。
程序映射地址：
返回值例子： mobe11.xml | mobe11.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
shopid 评论的商家编号 必填
orderid 商家账单编号 必填
goodsid 评论的商品编号 必填
commentlevel 评分 必填 1至5
content 评论的具体内容 必填
业务逻辑说明： 1.根据入参 将用户评论信息存入user_comment表。
备注：评分与评论类型对应关系：
1-2：差评
2-3：中评
4-5：好评
