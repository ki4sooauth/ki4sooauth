mobe03:根据接收到的LID信息，返回对应的信息
功能说明： 根据接收到的LID信息，返回对应的信息
程序映射地址：
返回值例子： mobe03.xml | mobe03.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
iphonetoken Iphone的token值 选填
ver 当前版本号 必填
业务逻辑说明： 一.主业务逻辑
1.手机端将“用户信息”传给平台；
2.如果“userid”，“sessionid”校验通过，平台将“对应的信息”返回给手机端。
二.支业务逻辑
1. 将lid相关信息发送到其他平台（例如lid范围增减人） 2. 将"用户信息"发给日志平台
