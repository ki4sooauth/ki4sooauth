mobl02:是否允许用户播放订单音频
功能说明： 是否允许用户播放订单音频。（用户通过手机结算新华书店电子商务网站上的订单时调用接口）
程序映射地址：
返回值例子： mobl02.xml | mobl02.txt
参数说明：
CODE NAME 是否必填 备注说明
orderid 订单编号 必填 6位年月+8位唯一标识+ 03
userid 用户名 必填
shopid 商家编号 必填
业务逻辑说明： 线上订单线下结算流程：
当用户线下pos机支付时，调用此接口，手机端接收到服务器端允许线下pos支付的消息后，下载生成的订单音频文件，并播放订单音频文件；转发器接收到有效音频信号之后，服务器分析出是查询订单请求，然后将订单信息返回给转发器，转发器将其发送给pos端，用户在pos端支付，线上订单线下结算流程结束，操作成功。
