mobb02:添加收藏
功能说明： 添加收藏
程序映射地址：
返回值例子： mobb02.xml | mobb02.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
adtype 收藏的信息的类型 选填 收藏的信息的类型：商品-G、活动-A、优惠劵-C当shopadid有值时此项必填
shopadid 信息的id 选填 信息ID和信息的URL二选一必填
adurl 信息的url 选填 商品、活动、优惠的访问地址;可通过解码此参数得到adtype信息ID和信息的URL二选一必填
业务逻辑说明： 1.根据userid、sessiondi校验登录状态；
2.根据adtype（商品-G、活动-A、优惠劵-C）、adurl、shopadid、userid将用户添加的收藏存入favorite_info。
