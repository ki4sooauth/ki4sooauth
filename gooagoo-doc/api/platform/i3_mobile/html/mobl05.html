mobl05:是否允许用户播放会员卡音频
功能说明： 是否允许用户播放会员卡音频 。 用户会员卡播放音频之前调用接口
程序映射地址：
返回值例子： mobl05.xml | mobl05.txt
参数说明：
CODE NAME 是否必填 备注说明
scardno 会员卡卡号 必填 6位LID（代表实体店）+8位唯一标识（userinfo表的usernum字段）
+会员卡类型（00关注卡，10～F0电子卡，F1～FF实体卡）
userid 用户名 必填
shopid 商家编号 必填
业务逻辑说明：
修改日志： 使用会员卡流程：
手机将会员卡信息提交至服务器，服务器确认会员身份并允许播放音频，根据手机端提供的会员卡信息 生成一个会员卡的音频文件，手机端将该音频文件下载下来，播放该发票音频文件，转发器接收到该音 频信号后，将其发送至服务器，服务器分析出是使用会员卡请求，然后将会员卡信息返回给转发器， 使用会员卡流程结束
