mobb08:查询商家精品
功能说明： 查询商家的精品信息
程序映射地址：
返回值例子： mobb08.xml | mobb08.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 选填 用户编号，唯一，通过UUID产生
sessionid sessionid值 选填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
pageindex 页码 必填 当pageindex=1，pagesize=-1时，不分页
pagesize 每页显示信息条数 必填
shopid 商家id 必填
业务逻辑说明： 1.根据shop查询商家的会员信息，获取关注数量、会员数量级会员卡卡头url 2.查询商家的精品信息。
备注：
3.只查询“未删除”的信息。
4.封装查询结果，返回数据。
