mobl04:是否允许用户播放取发票音频
功能说明： 是否允许用户播放取发票音频 。 用户取发票播放音频之前调用接口
程序映射地址：
返回值例子： mobl04.xml | mobl04.txt
参数说明：
CODE NAME 是否必填 备注说明
orderid 订单/账单编号 必填 6位年月+8位唯一标识+ 05
userid 用户名 必填
shopid 商家编号 必填
业务逻辑说明：
修改日志： 取发票流程：
手机将发票信息提交至服务器，服务器判断允许开据发票后，根据手机端提供的发票信息生成一个发票的音频文件，手机端将该音频文件下载下来，播放该发票音频文件，转发器接收到该音频信号后，将其发送至服务器，服务器分析出是取发票请求，然后将发票申请信息返回给转发器，转发器将开发票详情传给发票打印机或者PC端软件。播放成功后，在发票打印机上会打印出账单的发票。取发票成功。sp  发票打印机或者PC端软件。播放成功后，在发票打印机上会打印出账单的发票，取发票流程结束
