moba03:用户会员卡关联卡头信息标签
功能说明： 用户会员卡关联卡头信息标签
程序映射地址：
返回值例子： moba03.xml | moba03.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
ctimestamp 最大时间戳 必填 最后一次修改时间，同步全部则设置为0
业务逻辑说明： 一.主业务逻辑
1.手机端将“用户信息”、“最大时间戳”传给平台；
2.如果“userid”，“sessionid”校验通过，平台将该用户所有会员卡中“最后一次修改时间”大于“最大时间戳”的所有卡每张卡最新的一条记录返回给手机端。
二.支业务逻辑
将"用户信息"发给日志平台(是否需要“userid”，“sessionid”校验通过)
