moba10:用户申请电子会员卡
功能说明： 用户申请电子会员卡
程序映射地址：
返回值例子： moba10.xml | moba10.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址选填
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
shopid 店铺id 必填
realname 用户姓名 必填
sex 性别 选填 性别：M-男，F-女，P-其他
mobile 手机号 选填 手机号、身份证号二选一必填
idno 身份证号 选填 手机号、身份证号二选一必填
birthday 生日 选填 YYYY-MM-DD
address 地址 选填
postcode 邮编 选填
业务逻辑说明： 1.根据userid、sessionid校验登录状态；
2.根据入参，查询member_of_card表，判断此useid是否已经有了本商家的会员卡.如果有，则提示“用户已经是本商家的会员”，如果没有，继续下一步。
3.将信息添加到member_apply会员申请表中。
4.返回查询结果true/false。
