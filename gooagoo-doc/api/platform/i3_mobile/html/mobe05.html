mobe05:手机提交订单（点菜单、快速结账）
功能说明： 手机提交点菜单信息
程序映射地址：
返回值例子： mobe05.xml | mobe05.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填 用户编号，唯一，通过UUID产生
sessionid sessionid值 必填 手机登录后的sessionid值
mac手机mac地址 必填
takemethod提货方式 必填,0-直接拿走、1-前台提货、2-送货上门
mver手机程序版本选填
lidlid选填
entityposition实体店编号选填用户当前所在实体店编号
shopid 店铺id 必填
shopentityid 实体店编号 必填
goodssource 商品来源 0-表示来源购物车,其它表示不是来自购物车，当商品来自购物车，提交订单后，需把购物车中的商品清理掉
goodsidlist 商品列表 选填 格式如：[ { "goodsid": "00017STNPL7NIGF1T0000LEIISQNS0DL", "goodsnum":"2"}, { "goodsid": "00017STNPL7NIGF1T0000LEIISQNS0DL", "goodsnum":"20"}]
couponid 优惠凭证编号 选填
tableno2d 桌号二维码 选填 tableno2d 格式["gooagoo","00","实体店编号","桌号","房间号"] 当type=00时,此入参必填
address 收货地址 选填
name 收货人 选填
phone 联系电话 选填
type 订单类型 必填 00-餐饮业提交用户订单,01-餐饮业提交用户订单(远程点餐),11-零售业提交用户订单
业务逻辑说明： 用户选购完商品后将订单提交到服务器。
