mobk01:按日期查询用户消费记录
功能说明： 查询用户某天的消费记录信息。
程序映射地址：
返回值例子： mobk01.xml | mobk01.txt
参数说明：
CODE NAME 是否必填 备注说明
userid 用户id 必填
sessionid session值 必填
mac 手机mac地址 必填
date 查询日期时间 必填 查询时间（yyyy-mm-dd）
业务逻辑说明： 1.用户根据日期查询“当日订单”、“当日账单”、“当日活动”、“购物清单”的数目。查询“当日订单”、“当日账单”、“购物清单”的时候须校验用户登录信息，活动无需校验登录信息。若该天没有“当日订单”、“当日账单”、“当日活动”、“购物清单”，则返回“0”。
