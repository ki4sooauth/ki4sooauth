接口gtsd01:根据优惠劵编号查询商家优惠劵信息
功能说明: 根据优惠劵编号查询商家优惠劵信息
程序映射地址:
返回值例子: gtsd01.xml | gtsd01.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 转发器MAC地址 必填
shopentityid 实体店编号 必填
shopid 商家编号 必填
couponid 优惠凭证编号 必填 6位年月+8位唯一标识+ 01
业务逻辑说明： 手机端将优惠劵编号传到服务器端，服务器端同意使用此优惠劵后，根据此优惠劵编号生成一个音频文件，手机端在收到服务器端允许使用优惠劵后，将其音频文件下载到手机端，播放音频文件，转发器接收到音频信号，然后将音频号解析成号码发送至服务器，服务器分析出是使用优惠劵请求，然后将用户的商家优惠劵信息返回给转发器，转发器将接收到的商家优惠劵信息发送至pos机，pos机校验成功通过后，优惠劵使用成功。
