接口gtsc03:打印发票申请
功能说明: 打印发票申请
程序映射地址:
返回值例子: gtsc03.xml | gtsc03.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 转发器MAC地址 必填
shopid 商家编号 必填
shopentityid 商家实体店编号 必填
ctimestamp 最大时间戳 必填
业务逻辑说明： 一.主业务逻辑
1.转发器将“mac”、“shopid”、“shopentityid”、“ctimestamp”传给平台；
2.平台根据“shopid”、“shopentityid”将bill_invoice_log表中“c_time_stamp”大于“ctimestamp”的相关数据返回给转发器。
