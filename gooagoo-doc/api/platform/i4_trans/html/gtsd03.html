接口gtsd03:根据提货凭证编号查询商家订单/账单信息
功能说明: 根据提货凭证编号查询商家订单/账单信息
程序映射地址:
返回值例子: gtsd03.xml | gtsd03.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 转发器MAC地址 必填
shopentityid 实体店编号 必填
shopid 商家编号 必填
voucherid 提货凭证编号 必填 6位年月+8位唯一标识+ 04
业务逻辑说明： 手机将提货凭证提交给服务器端，服务器端允许用户提货后，根据提货凭证生成一个提货凭证的音频文件，手机端下载音频文件，并播放该音频文件，转发器接收到有效音频信号后，发送至服务器，服务器分析出是提货凭证使用请求，然后将订单信息返回给转发器，转发器将提货凭证信息发送给PC端，由服务人员给予提货。提货凭证的使用流程结束。
