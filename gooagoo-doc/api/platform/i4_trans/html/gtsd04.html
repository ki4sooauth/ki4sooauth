接口gtsd04:根据发票凭证查询发票申请信息
功能说明: 根据发票凭证查询发票申请信息
程序映射地址:
返回值例子: gtsd04.xml | gtsd04.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 转发器MAC地址 必填
shopentityid 实体店编号 必填
shopid 商家编号 必填
invoiceinfoid 开发票申请信息编号 必填 6位年月+8位唯一标识+ 05
业务逻辑说明： 手机将发票信息提交至服务器，服务器判断允许开据发票后，根据手机端提供的发票信息生成一个发票的音频文件，手机端将该音频文件下载下来，播放该发票音频文件，转发器接收到该音频信号后，将其发送至服务器，服务器分析出是取发票请求，然后将发票申请信息返回给转发器，转发器将开发票详情传给发票打印机或者PC端软件。播放成功后，在发票打印机上会打印出账单的发票。取发票成功。
