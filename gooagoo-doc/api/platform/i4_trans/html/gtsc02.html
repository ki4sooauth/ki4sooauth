接口gtsc02:订单查询
功能说明: gtsc02
程序映射地址:
返回值例子: gtsc02.xml | gtsc02.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 转发器MAC地址 必填
shopid 商家编号 必填
ctimestamp 最大时间戳 必填
shopentityid 实体店编号 必填
scardno 会员卡音频编号 选填
业务逻辑说明： 一.主业务逻辑
1.转发器将“mac”、“shopid”、“ctimestamp”、“scardno”、“shopentityid”传给平台；
2.如果“scardno”为空，则查询查询order_info表中订单状态为“0-用户提交”的订单信息列表。
（需通过order_info.order_id关联表order_detail_info查询相关信息）返回给转发器。
