接口gtsc18:商家查询重量确认
功能说明: 商家查询重量确认
程序映射地址:
返回值例子: gtsc18.xml | gtsc18.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 转发器MAC地址 必填
shopentityid 实体店编号 必填
ctimestamp 时间戳 必填
业务逻辑说明： 1. 转发器每隔一段时间，便请求平台，查看是否有“重量确认”的数据，平台接收到转发器的请求，将“重量确认”的数据返回给转发器，转发器拿到“重量确认”数据后，将其推送给第三方软件（餐饮软件），第三方获取“重量确认”数据后，开始下菜。 重量确认：针对店内以重量为计量单位的菜品给予确认。例如：顾客要求点一条二斤大的鲜鱼，但是店里没有二斤的鱼，后厨称了一条三斤重的鱼，此时店员助理需要征求顾客意见，是否可以。顾客若同意，则店员助理确认重量，通知后厨开始下菜。
