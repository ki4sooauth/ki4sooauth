接口gasc03:会员信息查询
功能说明: 通过会员卡音频卡号查询会员信息。
程序映射地址:
返回值例子: gasc03.xml | gasc03.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
scardno 会员卡音频编号 必填
shopuserid 店员登录账号 必填
业务逻辑说明： 一.主业务逻辑
1.通过“scardno”查询member_of_card表，获得userid、card_id（会员卡编号）。
2.根据userid查询user_profile表获取realname（真实姓名）。
3.根据card_id查询member_card表获取card_name（会员卡名称）。
4.然后根据shopid、userid查询integral_info积分表，获取useable_integral_number（可用积分）
5.将1、2、3、4的查询结果封装，返回结果。
