接口gasl12:重量确认
功能说明: 重量确认
程序映射地址:
返回值例子: gasl12.xml | gasl12.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
shopentityid 实体店编号 必填
tablename 餐桌号 必填
orderid 订单编号 必填
itemserial 自定义序列号 必填
goodsweight 菜品重量 必填
业务逻辑说明： 1.针对店内以重量为计量单位的菜品给予确认。例如：顾客要求点一条二斤大的鲜鱼，
但是店里没有二斤的鱼，后厨称了一条三斤重的鱼，此时店员助理需要征求顾客意见，
是否可以。顾客若同意，则店员助理确认重量，通知后厨开始下菜
