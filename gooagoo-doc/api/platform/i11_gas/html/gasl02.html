接口gasl02:查看全部订单
功能说明: 查看已开台的餐桌的用餐状态
程序映射地址:
返回值例子: gasl02.xml | gasl02.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
shopentityid 实体店编号 必填
pageindex 页码 必填
pagesize 每页信息显示条数 必填
tablename 餐桌号 选填 此参数有值时只返回此餐桌对应订单
业务逻辑说明： 一.主业务逻辑
1.平台根据“shopentityid”、“tablename”查询order_info表信息。
如果查询结果中scard_no(会员卡编号)不为空，则是"会员"，否则为"非会员"
2.平台根据“shopentityid”、“tablename”查询redis缓存中的数据
获取餐桌状态【0-空闲 1-已点餐 2-正在结账】。
3.组合数据返回给店员助理；
