接口gasd02:商品列表查询
功能说明: 商品列表查询
程序映射地址:
返回值例子: gasd02.xml | gasd02.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填 对应表shop_user_info中“user_id”
shopid 商家编号 必填
shopentityid 实体店编号 必填
pagesize 每页信息显示条数 必填
pageindex 页码 必填
categoryid 品类编号 选填
业务逻辑说明： 一.主业务逻辑
1.店员助理将“mac”、“shopuserid”、“shopid”、“shopentityid”、“pageindex”、“pagesize”、“categoryid”传给平台；
2.如果“categoryid”不为空，平台根据“shopid”、“shopentityid”查询“goods_category”表，以及“categoryid”对应所有“子类”信息；然后根据查询结果查询goods_base_info表
3.如果“categoryid”为空，则直接查询goods_base_info表所有商品信息；
4.封装查询结果，返回给店员助理。
