接口gasm02:发票申请
功能说明: 发票申请
程序映射地址:
返回值例子: gasm02.xml | gasm02.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
userid 用户编号 必填
orderid 订单编号 必填
shopentityid 实体店编号 必填
title 发票抬头 必填
shopid 商家编号 必填
invoicedtype 发票类型 必填 发票类型，0-个人，1-公司 必填
业务逻辑说明： 一.主业务逻辑
1.平台根据入参“orderid”查询order_info表相关信息
2.平台结合入参“title”及步骤1的查询结果，封装数据插入到bill_invoice_log（开发票申请信息）表;
3.返回bill_invoice_log中数据添加成功与否状态给店员助理；
