接口gase05:店内踪迹查询
功能说明: 查询用户历史上在实体店的足迹 商家---实体店 实体店---主区域 品牌---店内的区域
程序映射地址:
返回值例子: gase05.xml | gase05.txt
业务逻辑描述: 根据商家编号和位置id,查询下列历史足迹信息:会员在商家实体店停留情况、在商家实体店下的区域停留情况、在品牌各区域停留情况
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopUserId 店员ID 必填
userid 用户ID 必填
业务逻辑说明： 一.主业务逻辑
1.店员助理将“mac”、“billid”、“userid”传给平台；
2.平台根据“shopUserId”、“userid”查询下列历史足迹信息:会员在商家实体店停留情况、在商家实体店下的区域停留情况、在品牌各区域停留情况返回给店员助理。
