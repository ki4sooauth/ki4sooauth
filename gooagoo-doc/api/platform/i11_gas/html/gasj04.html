接口gasj04:店员扫桌号获取用户点菜单信息
功能说明: 店员扫桌号获取用户点菜单信息
程序映射地址:
返回值例子: gasj04.xml | gasj04.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
shopentityid 实体店编号 必填
pageindex 页码 必填
pagesize 每页信息显示条数 必填
tablename 桌号 必填
业务逻辑说明： 一.主业务逻辑
1.根据shopentityid、“tablename”查询order_info表中订单状态不等于“3-已结账”的数据，获取到order_id（订单编号）、bill_img(点菜单图片)、pay_price（订单金额，即合计）;
2.根据“shopentityid”、orderid在表order_detail_info查询goods_id(菜品编号)、goods_name(菜品名称)、pay_price（实际支付单间）、goods_num（菜品数量）等信息;
3.根据goods_id在表goods_base_info查询出goods_category_root（品类根节点）;
4.根据goods_category_root（对应goods_category中字段category_id-品类编号）在表goods_category查询出品类名称
5.组合查询结果返回给店员助理.
