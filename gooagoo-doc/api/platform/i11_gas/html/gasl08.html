接口gasl08:店员帮助用户起菜
功能说明: 店员帮助用户起菜
程序映射地址:
返回值例子: gasl08.xml | gasl08.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
shopid 商家编号 必填
orderid 订单编号 必填
shopentityid 实体店编号 必填
tablename 餐桌号 必填
itemserial 自定义序列号 选填 未填表示按整单起菜
goodsnum 菜品数量 选填
业务逻辑说明： 1.店员可帮助用户进行起菜操作。只有当起菜方式为“叫起”的时候，店员帮助用户提交起菜请求，第三方接收到请求，将其起菜信息打印，开始起菜。
