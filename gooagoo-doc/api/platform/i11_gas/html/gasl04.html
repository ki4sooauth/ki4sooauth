接口gasl04:已上菜品订单标记
功能说明: 标记已上菜品
程序映射地址:
返回值例子: gasl04.xml | gasl04.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
orderid 订单编号 必填
shopentityid 实体店编号 必填
tablename 桌号 必填
goodsinfo 已上菜品信息 必填 [ { "itemserial":
"00017STNPL7NIGF1T0000LEIISQNS0DL",
"goodsnum":"2"}]
业务逻辑说明： 一.主业务逻辑
1.平台将接收到的”goodsinfo“解析,得到菜品编号“goods_id”。
2.平台根据“goods_id”、“tablename”、“orderid”查询“order_detail_info”，修改“serve_num（上菜情况）”。 2.组合处理结果返回给店员助理.
