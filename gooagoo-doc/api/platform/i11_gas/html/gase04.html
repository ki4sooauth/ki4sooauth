接口gase04:消费记录明细查询
功能说明: 查询会员账单消费记录明细信息
程序映射地址:
返回值例子: gase04.xml | gase04.txt
业务逻辑描述: 根据会员ID及账单ID查询会员账单消费记录明细信息
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
pageindex 页码 必填
pagesize 每页显示条数 必填
orderid 订单编号 必填
业务逻辑说明： 一.主业务逻辑
1.店员助理将“mac”、“orderid”、“pageindex”、“pagesize”传给平台；
2.平台根据“orderid”在表“order_detail_info”中查询“goods_id”等信息；
3.平台根据“goods_id”在表“goods_base_info”、“goods_feature_info”中查询相关信息。
4.将1、2、3的查询结果封装，返回给店员助理。
