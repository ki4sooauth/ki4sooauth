接口gask02:菜品类别信息获取
功能说明: 店员查询菜品各类别根节点
程序映射地址:
返回值例子: gask02.xml | gask02.txt
参数说明：
CODE NAME 是否必填 备注说明
mac 店员助理MAC地址 必填
shopuserid 店员登录账号 必填
shopentityid 实体店编号 必填
pageindex 页码 必填
pagesize 每页信息显示条数 必填
业务逻辑说明： 一.主业务逻辑
1.根据“shopentityid”在表goods_category在查询所有一级品类信息-category_id（类别编号）、category_name(类别名称)、pic_url（类别实例图片url）;
2.组合数据返回给店员助理。
备注：
此处返回应该加一个节点，返回此商品是否有库存。零售业的商品查询要显示“库存量”，餐饮业的菜品查询要显示“有”或者“无”。
